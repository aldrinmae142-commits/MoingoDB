# JavaScript Destructuring Practice

This project demonstrates various aspects of JavaScript destructuring as required by the assignment criteria.

## Files and Criteria Mapping

### Criterion 1: Array Destructuring
**File:** `array-destructuring.js`
- Basic array destructuring
- Skipping elements
- Rest operator usage
- Default values in arrays

### Criterion 2: Object Destructuring
**File:** `object-destructuring.js`
- Basic object destructuring
- Destructuring with different variable names
- Partial destructuring
- Destructuring in function parameters

### Criterion 3: Nested Destructuring
**File:** `nested-destructuring.js`
- Nested object destructuring
- Nested array destructuring
- Complex nested structures
- Combining object and array destructuring

### Criterion 4: Default Values During Destructuring
**File:** `default-values-destructuring.js`
- Object destructuring with defaults
- Array destructuring with defaults
- Default values with undefined properties
- Function parameters with default destructuring

### Criterion 5: Renaming Variables During Destructuring
**File:** `renaming-variables-destructuring.js`
- Object destructuring with renaming
- Renaming with default values
- Nested destructuring with renaming
- Function parameters with renaming

### Criterion 6: Self-Assessment and Additional Practice
**File:** `self-assessment.js`
- Self-assessment questions and answers
- Additional complex examples
- Practical applications
- Self-check verification

## How to Run

1. Start the server: `node server.js`
2. Open `http://localhost:3000` in your browser
3. Open the browser console (F12) to see the output
4. Alternatively, run individual files with Node.js: `node array-destructuring.js`
